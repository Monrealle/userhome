# userhome — получить домашний каталог пользователя из passwd-файла

Утилита на bash, которая по имени пользователя находит его домашний каталог в файле `/etc/passwd` (или любом аналогичном файле).

## Синтаксис

userhome.sh [-f файл] [имя_пользователя]
Если `имя_пользователя` не указано, используется текущий пользователь (`$USER`).

## Коды возврата

| Код | Значение                                                                                                           |
|-----|--------------------------------------------------------------------------------------------------------------------|
| `0` | Успех. В stdout выведен только домашний каталог (без лишних символов).                                             |
| `1` | Пользователь не найден в указанном файле.                                                                          |
| `2` | Файл не существует / недоступен для чтения, или допущена синтаксическая ошибка (неверная опция, лишние аргументы). |

## Примеры использования

```bash
# Домашний каталог root из стандартного /etc/passwd
./userhome.sh root

# Домашний каталог текущего пользователя
./userhome.sh

# Поиск в альтернативном файле
./userhome.sh -f /tmp/test_passwd john

# Обработка ошибок
./userhome.sh unknown_user      # код 1, сообщение в stderr
./userhome.sh -f /no/file root  # код 2, сообщение в stderr

# Вызов справки
./userhome.sh --help
```

## Требования
- Bash ≥ 3.2.
- Стандартные утилиты: grep, cut, cat, echo.
